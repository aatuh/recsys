# Examples

The module ships runnable examples under `recsys-algo/examples/`.

## Install

```bash
go get github.com/aatuh/recsys-suite/api/recsys-algo
```

## Minimal example (popularity-only)

See `recsys-algo/examples/basic` for a runnable popularity-only pipeline.

## Full examples

- `recsys-algo/examples/personalized` — user profile + personalization
- `recsys-algo/examples/rules` — pin/boost/block rule application
