# Code of Conduct

This project adopts the **Contributor Covenant Code of Conduct**.

## Our pledge

We as members, contributors, and leaders pledge to make participation in our community a harassment-free
experience for everyone.

## Scope

This Code of Conduct applies within all community spaces and also applies when an individual is officially
representing the community in public spaces.

## Enforcement

To report incidents, contact: `conduct@recsys.app`.

## Full text

The full text of Contributor Covenant v2.1 is available at:
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
